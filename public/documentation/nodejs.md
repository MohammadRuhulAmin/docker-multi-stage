npm uninstall <package_name>

##  Install ESLINT:
- npm install eslint --save-dev


### ESLINT command:
- npx eslint . // to run on the whole project
- npx eslint <file_path> // to run a spicific file
- npx eslint <file_path> --fix // it will find error and fix automatically
